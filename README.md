# Nexus CRM + ATS — Backend API

FastAPI backend for Nexus CRM. Connected to Supabase PostgreSQL.

## Endpoints

- `GET /api/health` — health check
- `POST /api/auth/login` — login
- `POST /api/auth/register` — register
- Full API docs at `/docs`
